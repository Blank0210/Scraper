# Scraper

Follow the below steps in order to setup the project

1. Create a folder where you want to initialize the project
2. Open the folder in any code editor
3. Run - npm init -y in the terminal
4. Open the package.json and enter "type": "module",
5. Run - npm install cheerio in the terminal

In order to run the code make sure that scrape.js is in the newly created folder
Run - node scrape.js in the terminal.

This would display the output.
